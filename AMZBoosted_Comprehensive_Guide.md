## 🏗 Technical Overview (For Developers)
AMZBoosted uses a modern, **Service-Based Architecture** centered around a **Central Registry** (`lib/tool-definitions.ts`). This allows for:
- **Unified Logic**: Tools are defined once and work across Side Panel, Dashboard, and Schedules.
- **Lazy Loading**: UI components are loaded on-demand for maximum performance.
- **Safe Background Execution**: Isolated service layers ensure long-running tasks don't bloat the UI thread.
