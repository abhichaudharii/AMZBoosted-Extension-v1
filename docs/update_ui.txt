Here is the specific CSS to recreate that "Geometric Grid" background for your extension:
The "AMZBoosted" Grid Background Spec

1. Base Color: The underlying color is a deep, rich black (Zinc-950).

    Hex: #09090b

2. The Geometry Boxes (Grid Pattern): This is created using CSS gradients. It looks like a 1px thin line that is very transparent (about 3-5% opacity).

3. The Purple/Dark Glow: There is a subtle ambient glow (vignette) that makes the grid look like it's fading in and out.
Copy-Paste CSS

You can apply this to the <body> or main container of your extension popup/pages:
CSS

/* Apply this to your main container */
.amz-background {
  /* 1. Base Dark Color */
  background-color: #09090b;

  /* 2. The Geometric Grid Pattern */
  /* This creates vertical and horizontal lines every 40px */
  background-image: 
    linear-gradient(to right, #27272a 1px, transparent 1px),
    linear-gradient(to bottom, #27272a 1px, transparent 1px);
  
  /* Size of the squares */
  background-size: 40px 40px;

  /* 3. (Optional) Fading Mask */
  /* This makes the grid fade out at the edges so it's not too overwhelming */
  mask-image: radial-gradient(circle at center, black 40%, transparent 100%);
  -webkit-mask-image: radial-gradient(circle at center, black 40%, transparent 100%);
  
  /* Ensure it covers the whole area */
  width: 100%;
  min-height: 100vh;
}
